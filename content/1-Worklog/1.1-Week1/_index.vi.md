### Kiến thức đã học:

* **AWS Account Setup**: Cách tạo và cấu hình AWS Free Tier account
* **Kiro IDE**: Làm quen với Kiro - công cụ hỗ trợ phát triển với AI
* **FCJ Program**: Hiểu về cấu trúc chương trình First Cloud Journey
* **Team Collaboration**: Kỹ năng làm việc nhóm và giao tiếp
* **Discord Server Management**: Tạo và quản lý Discord server cho nhóm
* **Advanced JavaScript**: Các kiến thức JavaScript nâng cao để chuẩn bị cho dự án
* **Hugo Static Site Generator**: Cách sử dụng Hugo để tạo website tĩnh
* **Git & GitHub**: Quản lý version control và documentation
* **Draw.io**: Sử dụng Draw.io để vẽ sơ đồ kiến trúc và diagram
